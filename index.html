Sobre mim

Olá! Meu nome é Felipe Lunkes, mas pode me chamar de Lunx!

Sou um desenvolvedor backend pleno, atuando no desenvolvimento de projetos utilizando, entre outras tecnologias, Java, Spring Framework, Spring Boot, APIs RESTful, bancos de dados relacionais (como MySQL), Python, JavaScript/TypeScript, Docker, Linux, Redis, AWS SQS, Amazon S3, mensageria com Apache Kafka e Kinesis, testes unitários e de integração, arquitetura de microsserviços e AWS Lambda (serverless). Tenho experiência em Java, Assembly x86, C e Python, e estou me aperfeiçoando continuamente nestas linguagens e em Rust, JavaScript/TypeScript, HTML e CSS. Além disso, sou um entusiasta de programação baixo nível (Assembly x86) e de design e implementação de sistemas operacionais. No meu tempo livre, você poderá me encontrar jogando The Legend of Zelda: Tears of the Kingdom, The Legend of Zelda: Echoes of Wisdom, Ghostwire: Tokyo ou Minecraft, estudando, lendo ou programando em Assembly x86.

Hexagonix

O Hexagonix é um novo sistema operacional Unix-like, escrito do zero em linguagem Assembly x86. Ele é construído sobre o Hexagon, um núcleo (kernel) monolítico simples e leve, com suporte a modo protegido (32 bits), interface de dispositivos e sistema de arquivos FAT16 (leitura e escrita). Compatível com hardware antigo (1999 em diante) e atual.

Ver no GitHub

Lunix

Lunix é um sistema operacional em estágio inicial de desenvolvimento, escrito em C (com componentes escritos em Assembly x86) para a arquitetura x86. Possui fins educacionais, auxiliando no aprendizado da linguagem C, de tópicos de hardware e de arquitetura de sistemas operacionais. Possui suporte a initrd (initial ramdisk) e alguns dispositivos.

Ver no GitHub

Bin S.O

O Bin S.O é um sistema operacional feito como um hobby, para consolidar os estudos realizados em hardware e em Assembly x86 (16-bit). Ele é um sistema operacional simples, monotarefa e monousuário, composto de uma série de utilitários e um kernel bastante simples, com sistema de arquivos BinFS (desenvolvido para ele).

Run Ancient UNIX

O projeto visa possibilitar a execução (via emulação), de forma fácil e descomplida, de versões antigas do UNIX para PDP-11, VAX e x86 em hardware moderno, realizando o download e configuração das imagens, bem como a configuração do simulador e/ou gerenciador de máquinas virtuais (qemu). Desenvolvido utilizando shell script e Python.

freebsd-config

Projeto responsável por automatizar tarefas ligadas à instalação de pacotes não instalados por padrão em um sistema FreeBSD. O script permite instalar e configurar o servidor Xorg, instalar módulos de kernel das principais placas de vídeo e interfaces de rede, dentre outras funcionalidades. Desenvolvido utilizando shell script.

FilmPlay

O FilmPlay é um pequeno projeto desenvolvido em Java para obter informações de filmes utilizando a API do OMDb. Para obter as informações via solicitação HTTP, o FilmPlay utiliza a biblioteca GSON para serializar e deserializar o JSON enviado como resposta. As bibliotecas e build do aplicativo são gerenciados pelo Gradle.

O projeto visa exemplificar a criação de um currículo legível (para humanos e robôs) em Markdown, além da sua conversão para PDF.
