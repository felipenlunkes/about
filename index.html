Sobre mim

Olá! Meu nome é Felipe Lunkes, mas pode me chamar de Lunx!

Sou um desenvolvedor backend pleno, atuando no desenvolvimento de projetos utilizando, entre outras tecnologias, Java, Spring Framework, Spring Boot, APIs RESTful, bancos de dados relacionais (como MySQL), JavaScript/TypeScript, Docker, Linux, testes unitários e de integração, mensageria com Apache Kafka e Kinesis e arquitetura de microsserviços. Tenho experiência em Java, Assembly x86, C e Python, e estou me aperfeiçoando em Rust. Além disso, sou um entusiasta de programação baixo nível (Assembly x86) e de design e implementação de sistemas operacionais. No meu tempo livre, você poderá me encontrar jogando The Legend of Zelda: Tears of the Kingdom, estudando, lendo ou programando em Assembly x86.

Hexagonix

O Hexagonix é um novo sistema operacional Unix-like desenvolvido do zero, escrito em linguagem Assembly x86. Ele é construído sobre o Hexagon, um núcleo (kernel) monolítico simples e leve, desenvolvido para ser rápido e compatível com hardware recente e mais antigo.

Lunix

Lunix é um novo sistema operacional escrito em C para a arquitetura x86 para fins educacionais, servindo como campo de testes para meu aprendizado em linguagem C, hardware e arquitetura de sistema operacional. Alguns componentes são escritos em Assembly x86.

Ver no GitHub

Bin S.O

O Bin S.O é um sistema operacional feito como um hobby, para consolidar os estudos realizados em hardware e em Assembly x86 (16-bit). Ele é um sistema operacional simples, monotarefa e monousuário, composto de uma série de utilitários e um kernel bastante simples, com sistema de arquivos BinFS.

Run Ancient UNIX

O projeto visa possibilitar a execução (via emulação), de forma fácil e descomplida, de versões antigas do UNIX para PDP-11, VAX e x86 em hardware moderno, realizando o download e configuração das imagens, bem como a configuração do simulador e/ou gerenciador de máquinas virtuais (qemu).

freebsd-config

Projeto responsável por automatizar tarefas ligadas à instalação de pacotes não instalados por padrão em um sistema FreeBSD. O script permite instalar e configurar o servidor Xorg, instalar módulos de kernel das principais placas de vídeo e interfaces de rede, dentre outras funcionalidades.

FilmPlay

O FilmPlay é um pequeno projeto desenvolvido em Java para obter informações de filmes utilizando a API do OMDb. Para obter as informações via solicitação HTTP, o FilmPlay utiliza a biblioteca GSON para serializar e deserializar o JSON enviado como resposta. As bibliotecas e build do aplicativo são gerenciados pelo Gradle.

cv-exemplo

O projeto visa exemplificar a criação de um currículo legível (para humanos e robôs) em Markdown, além da sua conversão para PDF. Esse formato é útil para oferecer um currículo sucinto, sem distrações e que pode ser processado por mecanismos automatizados de sites de cadastro e análise de currículo.
