About me

Hello! My name is Felipe Lunkes, but you can call me Lunx!

I'm a backend developer with experience in developing solutions using Java (Spring Framework and Spring Boot), C# (ASP.NET and Entity Framework), and Python. I work on building scalable applications, focusing on event-driven microservices architecture, using RESTful APIs, relational databases (MySQL) and non-relational (MongoDB), Docker, Redis/Valkey, messaging with Apache Kafka and Kinesis, as well as AWS services (Lambda, S3, and SQS). I also have extensive practice in unit and integration testing.

I have solid experience in different languages and ecosystems, including Java, C#, Python, C, x86 Assembly, and shell scripts, as well as practical experience with Rust, Python, JavaScript/TypeScript, HTML, and CSS. I'm constantly evolving technically, with recent focus on Rust.


I'm an enthusiast of low-level programming (x86 Assembly) and interested in operating system design and implementation. In my free time, you might find me gaming, studying, reading, programming in x86 Assembly, or venturing into Rust. Additionally, I develop an operating system completely written in x86 Assembly, which can be found in my project Timeline.


Feel free to contact me and learn more about me!

DoeTech

DoeTech is a platform that aims to create an interface between companies and individuals interested in donating electronic equipment that would be discarded, and social institutions or people in need who wish to receive this type of equipment. The proposal of DoeTech is to reduce the production of electronic waste while favoring the reuse of electronic equipment by social institutions and low-income people. Developed in C# and Angular, using Entity Framework and ASP.NET MVC, it implements a clean architecture on top of the MVC model (MVC + Services + Repositories). It also implements a MySQL database, as well as support for storing images and documents using Azure Blobs Storage. Supports authentication via JWT (Web Token), sending notifications and emails, and password recovery on demand. Tested and validated on Azure hosting. DoeTech was awarded as an outstanding project carried out by students of the Systems Analysis and Development undergraduate course at the Pontifical Catholic University of Minas Gerais (PUC Minas), in the first semester of 2025.
